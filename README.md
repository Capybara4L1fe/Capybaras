# Capybara Paradise
A simple HTML site built to help me relearn the basics of web development, dedicated entirely to our favourite giant rodent—the capybara! 

## About the Project
This project is a work in progress. Right now, it features a clean layout with a few fun facts about capybaras. I plan to expand it later with more layout practice, styles, and interactive elements as I continue practicing my coding skills.

## Live Demo
* **[Link to Capybara Paradise](https://capybara4l1fe.github.io/Capybaras/)**

## How to Run It Locally
1. Clone or download this repository.
2. Open the `index.html` file directly in any web browser.
